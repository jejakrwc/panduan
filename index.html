Tentang Sistem

STOP LAMP adalah sistem lampu LED berbasis ESP8266 yang dikendalikan melalui Web Browser. Sistem ini digunakan untuk menampilkan teks, animasi, serta lampu sen, rem, dan hazard secara interaktif.

Mengakses Web UI

Sambungkan perangkat ke WiFi STOP_LAMP
Buka browser dan akses 192.168.4.1
Web kontrol akan tampil otomatis

Tab Tampilan

  • Masukkan teks yang ingin ditampilkan
  • Pilih efek animasi teks
  • Atur font, kecepatan, dan kecerahan
  • Pilih mode: Teks Animasi Kombinasi
  • Klik Terapkan untuk mengirim ke perangkat
